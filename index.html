  • Java, Spring 기반의 백엔드 시스템 구조 분석 및 간단한 수정 작업 수행

  • SVN, SQL 등 개발 환경과 협업 도구를 활용한 실무 환경 적응
  • 기능 개선, 오류 수정, 테스트 업무를 통한 실무 흐름과 개발 프로세스 체득

삼양식품의 창고관리시스템(WMS) 프로젝트에 투입되어 신규 화면 개발 및 테스트 업무를 수행하고 있습니다.

  • Nexacro(넥사크로)를 활용한 프론트엔드 화면 설계 및 기능 구현
  • 화면과 데이터베이스 간의 연동 테스트(조회 기능) 수행 및 관통 테스트 진행
  • 백엔드와 연동되는 API 호출 및 데이터 흐름 이해를 바탕으로 단위 기능 개발
  • 화면단에서 입력된 값을 백엔드로 전송하고, 결과를 화면에 출력하는 프로세스 구현
